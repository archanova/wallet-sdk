# Archanova Tutorial

## Installation

```bash
$ git clone https://github.com/netgum/archanova.git
$ cd ./archanova
$ npm run bootstrap
```

## Usage

Start first instance on `5100` port
```bash
$ npm run start:tutorial:1200
```

Start second instance on `5200` port
```bash
$ npm run start:tutorial:5200
```
